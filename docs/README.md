## ドキュメント更新ガイドライン

このドキュメントリポジトリは以下のガイドラインに従って更新してください：

1. **設計変更時**: 関連するドキュメントを更新し、必要に応じて新しいADRを作成
2. **ADRの作成**: 重要なアーキテクチャ決定を行う際は、必ず新しいADRを作成
3. **図表の更新**: システムの変更に伴い、図表も最新の状態に維持
4. **リンクの整合性**: 新しいドキュメントを追加する際は、このインデックスを更新
